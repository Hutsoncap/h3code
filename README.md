# H3 Code

H3 Code is an early-stage Electron app for agent-driven development.

This repository is a fork of [DP Code](https://github.com/Emanuele-web04/dpcode), which is itself a fork of [T3 Code](https://github.com/pingdotgg/t3code). H3 Code is being reshaped into a single "everything app" for development: terminal + browser + diff + git + agent (Codex and Claude) + editor + spaces, all in one window.

The upstream projects are still the foundation, but this repo now follows its own roadmap, product direction, and release process.

## Current status

This project is very early and changes quickly. Expect bugs, rough edges, and breaking changes while the Phase 0 and Phase 1 roadmap work lands.

If you want the source of truth for what the product is becoming, read [ROADMAP.md](./ROADMAP.md) first. The contributor rules and completion gates live in [AGENTS.md](./AGENTS.md).

## How to use

> [!WARNING]
> You need to have [Codex CLI](https://github.com/openai/codex) installed and authorized for H3 Code to work.

You can also just install the desktop app. It's cooler.

Install the [desktop app from the Releases page](https://github.com/Hutsoncap/h3code/releases)
